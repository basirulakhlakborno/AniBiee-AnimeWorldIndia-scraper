## 2026-02-24 - Static Page Verification
**Learning:** Static HTML files in `public/` are not covered by the existing `lint` or `test` scripts.
**Action:** Use custom Playwright scripts to verify visual changes and accessibility in static pages.

## 2026-02-24 - 404 Page Design Pattern
**Learning:** The 404 page uses a glassmorphism style (semi-transparent background with blur) for buttons.
**Action:** Reuse this style for other secondary actions on dark backgrounds to maintain consistency.
