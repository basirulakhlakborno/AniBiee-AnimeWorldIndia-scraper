## 2025-01-28 - Critical Readability on Disclaimers
**Learning:** Legal disclaimers in API landing pages are often set to illegible font sizes (7-9px), hindering accessibility for users with visual impairments.
**Action:** Always verify text size is at least 11px (preferably 12px+) even for secondary content like footers/disclaimers. Use `rem` units where possible for scalability.
