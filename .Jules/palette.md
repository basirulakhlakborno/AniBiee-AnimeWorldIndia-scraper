## 2026-03-11 - Prevent dead-end user flows on error pages
**Learning:** Error pages without clear navigation paths lead to dead-end user flows, causing frustration. Secondary actions on dark backgrounds benefit visually from the glassmorphism design pattern.
**Action:** Always include an explicit 'Return to Homepage' interaction path on error pages. Use glassmorphism (rgba background, backdrop blur, vertical translate hover) for secondary actions on dark backgrounds to maintain consistency and delight.
