## 2026-02-21 - Glassmorphism for Dark Themes
**Learning:** The 404 page utilizes a glassmorphism style (semi-transparent background with blur) for secondary actions like the "Return to Home" button.
**Action:** Reuse `.home-button` styles or similar glassmorphism patterns for non-primary actions on dark-themed pages to maintain visual consistency.

## 2026-02-21 - Preventing Dead-End Pages
**Learning:** Error pages without clear navigation trap users and increase frustration.
**Action:** Ensure all error pages include at least one prominent primary action to return to a safe state (e.g., "Return to Home").
