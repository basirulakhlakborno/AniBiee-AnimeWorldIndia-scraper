## 2024-05-22 - Semantic Structure and Interactive Elements
**Learning:** Static HTML pages often lack semantic landmarks (`<main>`) and interactive states on buttons, leading to poor accessibility and confusing user flows (dead ends).
**Action:** Always wrap primary content in `<main>` and ensure all interactive elements have visible focus/hover states and clear navigation paths.
