## 2024-03-04 - Prevent Dead Ends on Error Pages
**Learning:** Error pages (404, 403) on dark backgrounds lack clear call-to-actions, resulting in dead ends. Secondary UI actions on dark backgrounds need a consistent visual language to remain visible without overpowering the main content.
**Action:** Implemented glassmorphism style (rgba background + backdrop blur) for the return home button to provide a consistent, visually appealing CTA that aligns with the existing dark theme.
